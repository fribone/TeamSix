# 二手书流转系统
## 项目简介
本项目是基于 IntelliJ IDEA 开发的轻量级二手书流转管理系统，旨在解决广州软件学院学生/书籍爱好者之间二手书闲置、流转效率低的问题，提供书籍发布、检索、交易对接、状态管理等核心功能，降低书籍资源浪费，实现知识载体的循环利用。

## 核心功能
1. 基础功能：用户注册/登录、书籍信息发布（含分类、价格、成色标注）、书籍检索（按书名/作者/分类筛选）；
2. 核心流转：书籍收藏、交易留言、状态更新（待流转/已成交/已下架）；
3. 辅助功能：个人中心（我的发布/我的收藏）、简单的书籍评价。

## 开发环境
- 开发工具：IntelliJ IDEA 2025.3.3
- 核心技术：Java + Spring Boot + MySQL + MyBatis（基础版）
- 运行环境：JDK 1.8/17、Maven 3.8+
